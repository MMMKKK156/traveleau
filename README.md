# 🌏 旅行規劃助手

一個功能完整的旅行規劃 Web 應用程式，支援行程管理、地圖導航、分帳計算、待辦清單和翻譯功能。

## ✨ 主要功能

### 📅 行程管理
- **每日行程規劃**：依日期組織行程，支援時間、地點、分類
- **航班資訊**：記錄航班號碼、起降時間、航廈資訊
- **行程分類**：景點、美食、交通、住宿、購物、其他
- **導航功能**：一鍵導航至 Google Maps（自動帶入城市國家資訊）
- **附件管理**：支援圖片、PDF、連結附件
- **備註功能**：每個行程可加入詳細說明

### 🗺️ 地圖總覽
- 彙整所有行程的地點資訊
- 依日期分類顯示
- 點擊地點直接跳轉 Google Maps
- 自動統計每日地點數量

### 💰 分帳管理
- **成員管理**：新增/刪除旅遊成員
- **費用記錄**：記錄每筆消費的付款人和分攤成員
- **即時匯率**：自動獲取 TWD 對各國貨幣的即時匯率
- **智慧計算**：自動計算每人應付/應收金額
- **匯率轉換**：內建匯率計算器

### ✅ 待辦清單
- **提醒事項**：記錄重要事項（如簽證、保險）
- **行李打包**：建立打包清單，支援勾選完成
- **持久化儲存**：重新整理不會消失

### 🌐 翻譯捷徑
- **拍照翻譯**：快速開啟 Google 翻譯相機模式
- **雙向翻譯**：中文 ↔ 目標語言快速切換
- **自動適配**：根據選擇的城市自動設定目標語言

## 🚀 使用方式

### 開始使用
1. 開啟 `index.html` 檔案
2. 點擊 Header 編輯旅遊標題、日期
3. 系統會自動產生每日行程頁面

### 新增行程
1. 選擇日期（Day 1, Day 2...）
2. 點擊「+ 新增行程」
3. 填寫時間、名稱、地點、分類、備註
4. 儲存後可新增附件（圖片、PDF、連結）

### 使用導航
- **行程頁面**：點擊行程卡片上的地圖圖示
- **地圖頁面**：點擊地點名稱下方的位置連結
- 系統會自動組合「地點名稱 + 城市 + 國家」搜尋，提高精確度

### 分帳功能
1. 前往「分帳」頁籤
2. 新增旅遊成員
3. 記錄每筆消費（付款人、分攤成員、金額、幣別）
4. 系統自動計算結算金額

### 待辦清單
1. 前往「待辦」頁籤
2. 新增提醒事項或打包項目
3. 勾選完成項目
4. 資料會自動儲存

## 💾 資料管理

### 自動儲存
- 所有資料自動儲存至瀏覽器 localStorage
- 重新整理頁面不會遺失資料
- 支援多日行程規劃

### 匯出 / 匯入
- **匯出**：將資料匯出為 JSON 檔案備份
- **匯入**：從 JSON 檔案還原資料
- **清除**：清空所有資料回到初始狀態

### 資料包含
- 旅遊標題與日期
- 每日行程與航班資訊
- 行程附件（圖片、PDF、連結）
- 成員列表
- 消費記錄
- 提醒事項
- 行李打包清單
- 匯率設定

## 🎯 特色功能

## ⚠️ 注意事項

### 資料安全
- 資料儲存於本地瀏覽器
- 清除瀏覽器資料會遺失行程
- 建議定期匯出備份

### 網路連線
- 匯率更新需要網路連線
- 天氣資訊需要網路連線
- Google Maps 導航需要網路
- 其他功能可離線使用

### 檔案限制
- 附件圖片建議 < 5 MB
- PDF 檔案建議 < 10 MB
- 過大檔案可能影響效能

## 🔧 疑難排解

### 資料沒有儲存
- 確認瀏覽器允許使用 localStorage
- 檢查是否開啟無痕模式（無痕模式不保存資料）
- 嘗試重新整理頁面

### 匯率沒有更新
- 檢查網路連線
- 手動輸入匯率
- API 可能有請求限制

### 導航無法開啟
- 確認已輸入地點位置
- 檢查瀏覽器彈出視窗設定
- 使用地圖頁面查看所有地點

### 附件無法預覽
- 確認檔案格式正確（圖片：jpg/png，文件：pdf）
- 檢查檔案大小
- 嘗試重新上傳
